# Co-Parenting App Roadmap

## 🚀 MVP 1: Foundation & Core Calendar
**Цель:** Запуск основного функционала календаря, настройки опеки и базовых уведомлений.

| Feature Detail | Technical Detail | Difficulty | Priority |
| :--- | :--- | :--- | :--- |
| **Setup custody model at start** | Clear. Also %-wise.   | M | High |
| **Month first, week next, day third** |  No 3 days view.   | S | High |
| **Switch between "You" and "Him" view** | Clear. (In Day view, besides Week/Month) .   | — | High |
| **Have selected mom and dad at the same time** |  Show mutual views at the same time for Mom and Dad.   | M | High |
| **Event type - default + add your own** | Added filters predefined and manual.  Example: Filter out "School events" in Dec.   | L | High |
| **Reoccurrence** |  Clear.   | S | High |
| **Confirm pickup** | Other side sees it is picked up.  Clear.   | S | High |
| **Notifications** | e.g., 30 min or 1h before pickup.  Clear.   | M | High |
| **Holidays and vacations by country** |  Clear.   | S | High |
| **Add events only you can see** |  Related to switching views.   | S | High |
| **Sat/Sun to have diff color** |  Clear.   | S | High |

## 🛠 MVP 2: Communication, Receipts & Dashboards
**Цель:** Добавление инструментов для согласования изменений, финансового учета и улучшения UX через дашборды.

| Feature Detail | Technical Detail | Difficulty | Priority |
| :--- | :--- | :--- | :--- |
| **Receipts** |  Extra section.   | L | High |
| **Change requests as a special feature** |  Change option on events shown as notification and in summary dashboard.   | M | High |
| **Weekly summary (optional)** |  Dashboard of mutual activities for next week.   | M | High |
| **First screen updates** | Dashboard of last 5 changes that both parents can see (when other parent made them).  Click and open the event.   | L | Medium |
| **Structured chat – new message – change request** | Button for Change request.  Option to select new date, notification to other parent.   | M | Medium |
| **Attach image to the event** |  Clear.   | L | Medium |
