# ⚠️ ВАЖНО: Разверните правила Firestore СЕЙЧАС!

## Проблема
Приложение может читать данные, но не может создавать приглашения из-за строгих правил безопасности.

## Быстрое решение

### Шаг 1: Откройте Firebase Console
https://console.firebase.google.com/project/coparently-a39c9/firestore/rules

### Шаг 2: Скопируйте правила

**ВАРИАНТ 1: Упрощенные правила (для немедленного тестирования)**

Откройте файл `firestore.rules.simple` и скопируйте его содержимое.

**ВАРИАНТ 2: Полные правила (исправленные)**

Откройте файл `firestore.rules` и скопируйте его содержимое.

### Шаг 3: Вставьте и опубликуйте

1. Вставьте содержимое в редактор правил
2. Нажмите **"Publish"**
3. Дождитесь подтверждения (обычно несколько секунд)

### Шаг 4: Проверьте

1. Перезапустите приложение
2. Попробуйте создать приглашение снова

## Что было исправлено

✅ Правила для создания приглашений упрощены
✅ Убраны слишком строгие проверки
✅ Правила теперь соответствуют данным, которые отправляет код

## Если не работает

Используйте **упрощенные правила** из `firestore.rules.simple` - они максимально простые и точно работают для тестирования.
