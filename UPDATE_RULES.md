# 🔄 Обновите правила Firestore

## Что нужно сделать

1. **Откройте файл `firestore.rules.simple`** в корне проекта
2. **Скопируйте ВСЕ содержимое**
3. **Откройте Firebase Console:** https://console.firebase.google.com/project/coparently-a39c9/firestore/rules
4. **Вставьте содержимое** в редактор правил
5. **Нажмите "Publish"**

## Что изменилось

✅ Разрешена работа с коллекцией `events`
✅ Разрешена работа с коллекцией `child_info`
✅ Разрешено чтение профиля партнера (для отображения co-parent)

## После обновления правил

1. **Перезапустите приложение**
2. **Войдите в систему**
3. **Проверьте синхронизацию данных:**
   - Откройте Settings → Sync
   - Убедитесь, что данные синхронизированы
4. **Проверьте co-parent pairing:**
   - Откройте экран "Co-Parent Pairing"
   - Если у вас есть партнер, он должен отображаться

## Проверка, есть ли у вас партнер

Для отображения co-parent нужно:
1. ✅ Выполнить pairing (отправить/принять приглашение)
2. ✅ В вашем профиле должен быть `partnerId`
3. ✅ Данные должны быть синхронизированы с Firestore

Если после обновления правил co-parent все еще не виден:
1. Проверьте, выполнено ли pairing
2. Проверьте синхронизацию данных
3. Попробуйте выйти и войти заново
