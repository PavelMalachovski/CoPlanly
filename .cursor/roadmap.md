# 📅 Roadmap: CoPlanly — Android-приложение для совместного календаря родителей

CoPlanly — это минималистичное Android-приложение, основанное на **Google Calendar**, которое помогает разведённым или отдельно проживающим родителям координировать уход за ребёнком (или домашним животным 🐶), управлять расписанием и событиями, связанными с воспитанием.

---

## ⚙️ Технологии и стек

- **Frontend:** Kotlin + Jetpack Compose + Material3
- **Архитектура:** MVVM или Clean Architecture
- **Локальная база данных:** Room
- **Бэкенд:** Firebase / Supabase (авторизация, синхронизация, уведомления)
- **Интеграция:** Google Calendar API
- **CI/CD:** GitHub Actions (сборка APK)

## 📆 Этап 2. MVP — Локальный календарь (3–4 недели)

**Цель:** реализовать базовую функциональность без интеграции с Google Calendar.

### Функциональность:
- [x] Отображение календаря (Jetpack Compose + Material3)
- [x] CRUD-операции с событиями через Room (создание, редактирование, удаление)
- [x] Цветовая схема для родителей (например, “Мама” — розовый, “Папа” — синий)
- [x] Простая навигация (Navigation Component)
- [x] Хранение данных локально (без сети)
- [x] Поддержка смены месяца / недели в календаре

## 🔗 Этап 3. Интеграция с Google Calendar (3 недели)

**Цель:** синхронизировать события CoPlanly с Google Calendar.

### Функциональность:

- [x] Синхронизация событий (push/pull)
- [x] Отображение статуса синхронизации

## 👨‍👩‍👧 Этап 4. Совместное использование (4–5 недель)

**Цель:** реализовать совместный доступ к календарю между двумя пользователями.

### Функциональность:
- [x] Регистрация и вход через Firebase Authentication
